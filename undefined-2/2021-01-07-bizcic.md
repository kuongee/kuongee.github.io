* 객체 내부에 falsy한 값이 있을 때 falsy한 값을 제외한 객체를 만들어내라, JavaScript 배열 함수를 이용하여

```javascript
// 나중에 이렇게 다시 풀어봄
const testObj = {
  A: 'aaa',
  B: false,
  C: 'ccc'
};

let result = Object.entries(testObj).reduce((acc, curr) => {
  const [key, value] = curr;
  if(value) {
    acc[key] = value;
  }
  return acc;
}, {});

console.log(result);
```
