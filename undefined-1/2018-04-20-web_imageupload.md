### 1. multer 사용

* multer 사용법 [https://github.com/expressjs/multer](https://github.com/expressjs/multer)

### 2. Database

* MariaDB 사용

#### table 구성

```text
테이블명: image
스키마: id, path
```

#### table 적용

image 테이블에는 이미지의 제목\(path\)만 저장하고 개별 아이디 번호를 부여

이미지의 제목은 앞부분에 타임스탬프를 덧붙여 저장 → multer에서 처리

### 3. 목록 보이기
